# Excel Sales Dashboard & Data Analytics Project

## Overview

This project is an **Excel-based Data Analytics and Business Intelligence dashboard** that analyzes sales performance, profit trends, product categories, and regional insights using advanced Excel features.

It includes interactive dashboards, pivot tables, charts, and formulas to transform raw data into meaningful business insights.

---

## Project Objective

The main objective of this project is to:
- Analyze sales and profit data
- Create interactive dashboards using Excel
- Apply advanced Excel formulas and functions
- Build Pivot Tables and Pivot Charts
- Present data in a clear and meaningful way for decision-making
